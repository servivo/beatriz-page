Red Wine Trail Page

The first coding project I designed during my SheCodes Basics course about HTML and CSS. It is an informative website about the Red Wine Trail (Rotweinwanderweg) in Rheinland-Pfalz, Germany. As Bootstrap and Flexbox were not part of the course, I taught myself how to design a navigation bar with multiple pages and how to place photos and texts side by side. The photos on the website are all taken by me.

Weather App

Web App which works with a Weather Forecast API Open Weather API to be able to display the weather forecast in different cities. It is built with React. The search icon fetches the users' location through their IP and displays the real time temperature and forecast. To challenge myself, I decided to include the Google Places API, which suggests the names of the cities as you start typing them in the search bar. I used my favourite colors (red and black), to differentiate myself from other projects in light colors.

Dictionary App
